摄像头成功连上了酒店的web认证网络

为了实现家里和这边的互动，在深圳家里安装了360小水滴后，我也带了一个摄像头来迪拜。但是到了迪拜后，因为一直住在酒店，酒店WiFi网络使用web认证系统，但是摄像头不支持web认证，只支持密码认证，摄像头一直闲置。直到今天，才找到解决方案。

使用自带AP创建一个自己的网络
因为酒店WiFi信号不佳，刚好自己一直带着三年前买的TP-LINK火柴盒3G router，一番搜索后找到了网线接口，将网线转换为AP后，发现支持密码，简单设置完成后，使用自己电脑/手机通过AP登陆完成web认证，此时整个AP就完成接入了。后续任何终端，包括摄像头再接入自己创建的这个AP，都不再需要Web认证了。

赶紧发个消息给媳妇，让她监督下。
